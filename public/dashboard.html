System Settings

Production/Sandbox Mode & Bank Endpoint Configuration
Transmission Mode

PRODUCTION: NACHA files and wire instructions are transmitted to the configured external bank endpoint via HTTPS. No self-transmit.
SANDBOX: Files are transmitted internally (self-transmit + auto-accept) for testing.

External Bank Endpoint

Configure the receiving bank's HTTPS endpoint. In Production mode, all ACH and wire transmissions route here via the Open Bank REST API.

Bank Registry (Quick Setup)

Select a pre-configured bank template to auto-fill endpoint details. You can customize after applying.

Routing Decision Logic

Payment Amount < $25,000: Routes to ACH Pipeline (batch processing)

Payment Amount ≥ $25,000: Routes to Wire Transfer (real-time Fedwire)

Urgent flag set: Forces Wire Transfer regardless of amount

Production Mode: ACH → External bank endpoint over HTTPS | Wire → External wire endpoint

Sandbox Mode: ACH → Self-transmit + auto-accept | Wire → Internal processing
